Worksheet Exercise 3.

Let \(T \colon \mathbb{R}^5 \to \mathbb{R}^5\) be a linear map.

  1. Prove that it is not possible that \({\rm range}\, T = {\rm null}\, T.\)

  2. Show, by a specific example, that the above result is possible for a linear map \(T \colon \mathbb{R}^4 \to \mathbb{R}^4.\)
